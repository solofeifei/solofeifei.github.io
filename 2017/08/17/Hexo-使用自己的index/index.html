Hexo 使用自己的index

SOLO
2017-08-17
Hexo

开题

玩儿GitHub很久了，但一直没有自己的主页和博客，感觉有点儿脱轨。最近有点儿闲来无事，折腾了一下GitHub Page。没用有官方的教程，用的是Hexo，简单方便。虽然是码农、搬砖的，但对美和个性也有那么一丝丝的追求。苦于不会前端，于是东拼西凑、左问右问搞出了一个自己还算满意的首页（index页）。想用自己的index+Hexo模式。但是用过Hexo（或其他）的人都知道，index页是模版自动生成的，而且每次编译（hexo g）都会更新。好不容搞到了一个主页，还不能和Hexo模版搭配使用，难过。。。但也不甘心。

问题：怎么才能在Hexo模版中使用自己的主页呢？

解决方式一：自己搞一套模版。声明，自己不会前端，想都没敢想。
解决方式二：看Hexo模版源码，由于做开发的，总想着index.html是在哪儿怎么生成的，修改一下名 字，只要不是index就可以了。这种方式是对的，但是Google、百度都没找到那段代码。
解决方式三：经历方式二后，眼睛都花了，但是突然看了一个配置文件，能修改index.html路径。心里美啊。。。只要把Hexo模版生的index路径修改成二级目录（只要不是在根目录下），GitHub就不会加载它了，然后再把自己index放到根目录下，哈哈大功告成。来看看步骤：
